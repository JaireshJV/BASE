Predominantly Used HTTP Status Codes (Developer View)

2xx – Success

Status Code Definition Description
200 OK Request successful and response contains data
201 Created Resource successfully created (POST requests)
204 No Content Request successful but no response body

3xx – Not Modified

Status Code Definition Description
304 Not Modified Requested data will give constant data

4xx – Client Errors

Status Code Definition Description
400 Bad Request Invalid request payload or parameters
401 Unauthorized Authentication required or failed
403 Forbidden Access denied despite valid authentication
404 Not Found Requested resource does not exist
409 Conflict Request conflicts with current server state

5xx – Server Errors

Status Code Definition Description
500 Internal Server Error Generic server-side error
502 Bad Gateway Invalid response from upstream server
503 Service Unavailable Server is overloaded or down for maintenance
504 Gateway Timeout Upstream server failed to respond in time
