# Check the versions 

1) node     ==> node -v
2) npm      ==> npm -v
3) expo-cli ==> expo --version

# Installations 

1) expo-cli ==> npm install -g expo-cli

# Creating APP 

(i) npx create-expo-app MyApp 
(ii) cd MyApp

# Running the App 

(i) npm start 

# Mobile View Setup

(i) Install ExpoGo App from PlayStore .
(ii) Scan the QR Code which generated in the terminal .
